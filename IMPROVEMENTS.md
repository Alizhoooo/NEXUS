# Улучшения Nexus Platform

## 🎨 Что было добавлено и улучшено

### 1. **Тёмная тема (Dark Mode)** ✨
- Автоматическое определение системной темы
- Переключатель темы в хедере (иконки солнца/луны)
- Сохранение предпочтений в localStorage
- Плавные переходы между темами
- Полная поддержка всех компонентов

**Файлы:**
- `index.html` - добавлена конфигурация darkMode и CSS переменные
- `App.tsx` - логика переключения темы, состояние isDarkMode

### 7. **Тёмная тема для Dashboard**
Все карточки статистики и графики теперь поддерживают тёмную тему:
- Тёмный фон карточек (`dark:bg-dark-card`)
- Контрастные границы (`dark:border-dark-border`)
- Цвета текста адаптируются (`dark:text-slate-100`)
- Иконки с правильными цветами (`dark:text-primary-400`)

## 🚀 Как использовать

### Переключение темы
```tsx
// Вручную через кнопку в хедере
// Или программно:
document.documentElement.classList.toggle('dark');
localStorage.setItem('darkMode', 'true');
```

### Добавление тёмной темы в новый компонент
```tsx
<div className="bg-white dark:bg-dark-card 
                text-slate-900 dark:text-slate-100
                border-slate-200 dark:border-dark-border">
  {/* контент */}
</div>
```

### Использование cn() утилиты
```tsx
import { cn } from './lib/utils';

className={cn(
  'base-class',
  isActive && 'active-class',
  className
)}
```
